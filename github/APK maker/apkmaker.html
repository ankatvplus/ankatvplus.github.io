📦 APK Maker Studio

URL veya HTML kodu ile Android Studio projesi oluşturun → Kendi APK'nızı derleyin

⚡ WebView ile siteyi gösteren bir Android projesi oluşturulur. İnternet izni eklenir, JavaScript etkindir.
🎨 HTML kodunuz assets klasörüne eklenir ve WebView ile gösterilir. CSS, JS, resimler (base64 veya local) çalışır.
💡 Hazır. "Android Projesi Oluştur" butonuna bas → ZIP indir → Android Studio'da aç → APK üret.
📘 APK yapmak için adım adım rehber
  1. Bu araç: URL veya HTML kodunu alır, tam çalışan bir Android Studio projesi (Java + WebView) oluşturur.
  2. ZIP dosyasını indirin ve herhangi bir klasöre çıkarın.
  3. Android Studio'yu açın → "Open" → çıkardığınız klasörü seçin (AppMakerProje klasörü).
  4. Gradle senkronizasyonunu bekleyin (internet gerekir).
  5. Menüden Build → Build Bundle(s) / APK(s) → Build APK(s) seçeneğine tıklayın.
  6. İşlem tamamlandığında sağ altta "locate" diyerek app/build/outputs/apk/debug/ içindeki app-debug.apk dosyasını alın.
  7. APK'nız hazır! İmzalama için release build de üretebilirsiniz.

✅ Bu yöntemle %100 çalışan, istediğiniz URL veya HTML içeriğini gösteren bir Android uygulaması oluşturup APK elde edersiniz.
